## Developed With

- [Babel 7.14](https://babeljs.io/) - A transpiler for javascript
- [Webpack 5.x](https://webpack.js.org/) - A module bundler
- [SCSS](http://sass-lang.com/) - A css metalanguage

---

### Install and run

- Install node modules

  ```bash
  cd react-starter
  npm install
  ```

  - run webpack

  ```bash
  npm start
  ```

---

#### Build

- build project in dist folder

```bash
npm run build
```
